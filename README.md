# CodeWars

- 非常好的刷题平台，面向测试用例编程，`cmd + s` 即可跑测试用例
- 题目有趣味且实用，习得的编码技巧很容易用到日常工作中
- 利用筛选条件例如【正则相关 + 好评度 + 难度】，能较为全面的掌握一个方向的知识
- 完成后发现更好的答案，重写一遍，有利于自己的编码习惯国际化
